# 🌐 Niya Sharma — Personal Portfolio

A clean, responsive personal portfolio website built with HTML, CSS, and Bootstrap 5.

## 🔗 Live Demo

> https://niyasharma25.github.io/portfolio

---


## ✨ Features

- **Responsive** — works on mobile, tablet, and desktop
- **Sticky Navbar** — glassmorphism effect with smooth scroll & active link highlight (Bootstrap Scrollspy)
- **Hero Section** — name, role, and quick links to GitHub & contact
- **Skills Section** — tech pill badges + animated progress bars
- **Contact Section** — email, GitHub, and LinkedIn cards + a message form
- **No JavaScript** — interactivity handled entirely by Bootstrap's data attributes

---


## ✏️ Customisation

| What to change | Where |
|---|---|
| Name, role, description | `index.html` — Hero section |
| Skills & progress bars | `index.html` — Skills section |
| Contact links | `index.html` — Contact section |
| Colours & fonts | `style.css` — `:root` variables |
| Add projects back | Add a `#projects` section in `index.html` |
